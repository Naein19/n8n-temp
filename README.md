\## 📦 Installation \& Usage



\### 1. Import Workflow

\- Open n8n

\- Navigate to \*\*Workflows\*\*

\- Click \*\*Import\*\*

\- Upload `.json` file



\### 2. Configure

\- Set credentials (API keys, OAuth, etc.)

\- Update environment variables

\- Adjust nodes if required



\### 3. Execute

\- Run manually OR trigger via webhook / cron
